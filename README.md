# STM32F103C8T6 / SPI
Example of SPI1 communication stm32f103c6t8 microcontroller.

# SPI1 Pinout
| PORT  | SPI1 | SPI2 |
| :---: | :---: | :---: |
| SS | PA4 | PB12 |
| SCK | PA5 | PB13 |
| MISO | PA5 | PB14 |
| MOSI | PA6 | PB15 |
