# STM32F103C8T6 / SPI
Example of SPI1 communication of stm32f103c6t8.

# SPI1 Pinout
| PORT  | SPI1 | SPI1 remap |
| :---: | :---: | :---: |
| SS | PA4 | PB2 |
| MOSI | PA5 | PB3 |
| MISO | PA6 | PB4 |
| SCK | PA7 | PB5 |
